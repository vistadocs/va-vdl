 How do I get the latest version of VA FileMan?
Download it right here!  Here is FileMan Version 22, with patches through October, 2004, already included. This is a ZIP file that unzips to a file of 797 routines, "fm22.rsa", in Cache' format. 
 

How do I install it?
Follow the instructions.
 

Is Version 22 backwards-compatible?

Yes. But a few of the VA applications packages were relying on undocumented variables and behaviors.  They have had to be repaired.  If you are running Lab/Blood Bank, you will need patches LR*5.2*212 and LR*5.2*221.  If you are running Radiology, you will need patch RA*5*7.  If you are running the Dental package, Patch DENT*1.2*26 will correct a minor user annoyance. 

Does Version 22 require 1995 Standard MUMPS?

Yes!  Please note that VA FileMan Version 22 will not run under operating systems which do not support at least the following 1995 MUMPS constructs: 

  • Two-argument $ORDER and $GET functions
  • MERGE command
  • 'Alternative' pattern-match syntax
  • $NAME, $REVERSE, $QUERY, $QLENGTH and $QSUBSCRIPT functions


Can I still get Version 21?

Yes. Just click  right here.
 

What about Y2K?

FileMan is Y2K compliant.  Be sure you have the latest copy of the FileMan date routines. 

How can I make FileMan understand 1995 Standard M[UMPS]?

Please note the "DIM" patch.  It is a part of Version 22, and can be added to any version of FileMan.  It is an upgrade to the DIM* routines (which parse MUMPS input to FileMan), which has been released by the VA. Click on DIM* patch at left.
